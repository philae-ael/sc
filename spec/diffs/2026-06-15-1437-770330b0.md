## Diff

Add new command `sc uncommit`:

- Runs `git reset --soft HEAD~1` in the current working directory.
- Undoes the last commit while keeping its changes staged in the index/worktree.
- Returns a `GitError` `Result` on failure (no working tree, no parent commit, etc.).
- Prints `✓ Last commit undone` on success.
- Registered in `src/cli.ts` like other commands; implementation lives in `src/commands/commit.ts` using a new `softResetToParent` helper in `src/utils/git.ts` (or reuses existing reset helper pattern).
