# Add --amend flag to sc commit command

- ID: 3dcfd816
- Created: 2026-06-15T14:21:29.860Z
- Target: spec/technical-spec.md
- Status: pending incorporation

## Diff

Update the `sc commit` command to support an optional `--amend` flag:

- `sc commit <message>` / `sc c <message> [--amend]`
  - When `--amend` is provided, amends the last commit with the new message instead of creating a new commit.
  - When `--amend` is not provided, behavior remains unchanged: stages all changes, squashes staging commits if present, and creates a normal commit with the provided message.
  - Prints appropriate feedback when no changes exist.
